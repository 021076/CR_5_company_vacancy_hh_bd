# CR_5_company_vacancy_hh_bd

Программа, которая будет
- получать информацию о работодателях и вакансиях с платформы hh.ru
- сохранять полученные данные в базу данных
- работать с базой данных через СУБД PostgreSQL (библиотека psycopg2)

Программа состоит из 2-х классов:
1. Класс APIhh для работы с платформой hh.ru
   для подключения к API и получения данных о работодателях и вакансиях по id работодателя.
2. Класс для работы с базой данных DBManager.
   В классе реализованы такие методы как создание, удаление базы данных и таблиц, наполнения таблиц,
   получение информации о базе данных, получение данных (выборки) о работодателях и вакансиях из БД

Интерактивность обеспечивается функцией interface.

Запуск программы выполняется через функцию main
